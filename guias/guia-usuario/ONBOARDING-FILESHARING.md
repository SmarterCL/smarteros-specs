# File Sharing y Onboarding con Dokploy y VPS

## File Sharing
Cada usuario puede compartir archivos usando Dokploy, que permite transferencias seguras entre el entorno local y el VPS. El servicio de file sharing está documentado en la sección de infraestructura y puede ser activado por usuario desde el panel de onboarding.

### Pasos para compartir archivos:
1. Acceder al panel de usuario en Dokploy.
2. Seleccionar la opción "Compartir archivo".
3. Elegir el archivo y destino (VPS o usuario).
4. Confirmar la transferencia.

## Onboarding de Agentes y RAG
Para cada usuario, se crea un espacio dedicado para pruebas de agentes y RAG (Retrieval-Augmented Generation). El onboarding incluye:
- Creación de tablas y espacio de embeddings.
- Lanzamiento de un agente de prueba en caliente.
- Acceso a la tienda Shopify y Odoo para pruebas de integración.

### Proceso de onboarding:
1. Registro del usuario.
2. Asignación de espacio RAG y agente de prueba.
3. Acceso a integraciones (Shopify, Odoo).
4. Monitoreo y métricas de uso.
