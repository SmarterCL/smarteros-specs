# Espacio RAG y Agentes de Prueba

Cada usuario tiene acceso a un espacio RAG (Retrieval-Augmented Generation) dedicado, con tablas de embeddings y agentes de prueba en caliente para onboarding.

## Proceso
1. Creación de tablas de embeddings por usuario.
2. Lanzamiento de agente de prueba (multi-turn, RAG).
3. Métricas y monitoreo de uso.
