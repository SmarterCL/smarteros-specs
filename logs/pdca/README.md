# Sistema PDCA - CV Operativo

## Regla del Sistema

Este sistema registra ciclos reales de Plan-Do-Check-Act como evidencia de aprendizaje operativo.

### Estructura de un Ciclo PDCA

Cada archivo en este directorio debe seguir la estructura:

```
# Fecha - Integración

## Plan
Qué se pretendía lograr

## Do
Qué se ejecutó

## Check
Qué resultados se obtuvieron

## Act
Qué se aprendió y cómo se ajustará

## Decisión Tomada
Acción concreta para el próximo ciclo
```

### Principio
- Solo se registra lo que ya funcionó
- El error es señal, no excepción
- La fricción es información
- La evidencia es el sistema funcionando
