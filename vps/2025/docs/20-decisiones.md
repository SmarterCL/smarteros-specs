# 20-decisiones.md - Decisiones 2025

## Decisiones clave tomadas en 2025

### Enero 2025
- Separación estricta entre MCP e infraestructura
- Eventos como única fuente de verdad
- Métricas calculadas desde eventos reales

### Febrero 2025
- Implementación de contratos estables para integraciones
- Definición de skills como unidades de lógica independientes
- Establecimiento de puntos de control MCP

### Marzo 2025
- Validación de arquitectura con partners
- Ajuste de métricas clave para dashboard operativo
- Establecimiento de protocolos de seguridad

### Abril 2025
- Establecimiento de versionado semántico para specs
- Definición de proceso de auditoría MCP
- Implementación de sistema de rollback seguro
