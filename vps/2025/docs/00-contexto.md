# 00-contexto.md - Contexto 2025

## Estado del proyecto SmarterOS al inicio del 2025

- Versión inicial: v3.0
- Objetivos del año: Estabilidad operacional y expansión de MCP
- Stakeholders clave: Equipo interno, partners, clientes piloto
- Recursos disponibles: Infraestructura en producción, equipo de desarrollo
- Restricciones: Presupuesto limitado, dependencias externas

## Alcance definido

- MCP (Micro Control Plane) como núcleo operativo
- Integraciones con sistemas externos (ACH, SUSESO, SII)
- Dashboard operativo para monitoreo
- Sistema de skills extensible
