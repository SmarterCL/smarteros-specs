# VPS - Versionado, Proceso y Seguimiento

## Cómo leer este VPS

Este directorio contiene la historia operativa y ejecución del proyecto SmarterOS, organizada por años.

## Estructura

- `/2025/` - Historia del 2025
  - `specs/` - Snapshots usados en producción
  - `docs/` - Documentación explicativa del año
  - `ops/` - Bitácoras, decisiones e incidentes
  - `releases.md` - Registro de liberaciones

- `/2026/` - Planificación y evolución para 2026
  - `specs/` - Especificaciones en desarrollo
  - `docs/` - Documentación del año en curso
  - `ops/` - Operaciones y ejecución
  - `roadmap.md` - Planificación anual

## Relación con el repositorio

- **Repositorio GitHub** (smarteros-specs) → estado vigente de las especificaciones
- **VPS** → historia de versiones usadas, cuándo y por qué

## Principio operativo

> El VPS guarda historia y ejecución. El repo guarda contratos. No mezclar.

## Regla dura

Un año no se reescribe. Solo se agrega contexto o cierre.
